==>This project is based on Operating Systems concepts and focuses on reducing CPU energy consumption using Dynamic Voltage and Frequency Scaling (DVFS).

📌 Overview

The project implements a CPU scheduling approach that adjusts frequency and voltage based on workload.
This helps in saving energy while still maintaining good system performance.

🎯 Objective

Reduce power consumption
Maintain efficient CPU performance
Implement energy-aware scheduling

⚙️ Working

Processes are scheduled by the CPU
DVFS adjusts frequency and voltage
Execution happens with optimized energy usage
Energy and performance metrics are calculated

📉 Energy Formula

Energy ∝ Voltage² × Frequency × Time

✨ Features

DVFS-based scheduling
Energy-efficient execution
Performance tracking

🧑‍💻 Run
gcc main.c -o scheduler
./scheduler
